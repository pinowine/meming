## 🇬🇧 English

**Meme-imize your language!**

BadWriter is a hilarious and interactive project designed to automatically rewrite your text—making it more interesting, slang-heavy, or simply... worse! Write like an IELTS 3.0 student and embrace the chaos of the mixture of different linguistic styles.

### Project Versions & Branches

This repository hosts two distinct versions of the project, maintained on separate branches:

1.  **[Browser Extension Version](https://github.com/pinowine/meming/releases) (extension Branch)**
    - A **Chrome** extension that allows you to rewrite text directly on any webpage.
    - **Features:**
      - Hold `Alt` + Click on hovered text to instantly replace it with a meme-ified version using a brutalist typewriter effect.
      - Customize rules and view some of the built-in replacement rules via Popup and Side Panel UI.

2.  **[Web Application Version](https://badwriter.ibuprofennist.com/) (main Branch)**
    - A standalone web interface.
    - Receiving news keyword and scraping data from [The Guardian Open Platform](https://open-platform.theguardian.com/) to generate a meme-ified version of the news.
    - Also allows users to copy and paste their own text to be rewritten.

### How to play

#### Browser Extension

1. Download the extension (**NOT** the source code!) from the [new releases](https://github.com/pinowine/meming/releases).
2. Install it in Chrome by dragging the zip file to [`chrome://extensions/`](chrome://extensions/).
3. Allow the permission to access the current tab.
4. Click the extension icon in the toolbar to open the popup or right click the extension icon to open the side panel and customize the settings.
5. Have fun!

#### Web Application

1. Go to [the hosting website](https://badwriter.ibuprofennist.com/).
2. Nothing else. Just have fun!

## 🇨🇳 中文

**梗语生成器！**

> [!NOTE]
> 该项目目前只为英文语法开发，因此不会在默认情况下对其他语言有任何更改。

BadWriter 是一个仅仅为了好玩的互动项目，旨在自动重写你的文本——让它变得更有趣、充满黑话，或者干脆……写得更糟！写出雅思 3.0 水平的文本，拥抱将不同语言模式糅杂的一片混乱吧。

### 项目版本与分支

这个仓库目前包含两个分支，分别是：

1.  **[浏览器扩展版本](https://github.com/pinowine/meming/releases) (extension Branch)**
    - 一个 **Chrome** 扩展，允许你在任何网页上直接重写文本。
    - **功能:**
      - 按住 `Alt` + 点击悬停文本，即可使用打字机效果将其替换为 meme 版本。
      - 通过 Popup 和 Side Panel UI 管理自定义规则，并可以浏览内置的一部分替换规则。

2.  **[网页应用版本](https://badwriter.ibuprofennist.com/) (main Branch)**
    - 一个独立的网页应用。
    - 接收新闻关键词并从 [The Guardian Open Platform](https://open-platform.theguardian.com/) 抓取数据，生成 meme 版本的新闻。
    - 也允许用户复制和粘贴自己的文本进行重写。

### 快速开始

#### 浏览器扩展

1. 从 [new releases](https://github.com/pinowine/meming/releases) 下载扩展（**不是**源代码！）。
2. 将 zip 文件拖到 [`chrome://extensions/`](chrome://extensions/) 以在 Chrome 中安装。
3. 允许访问当前标签页的权限。
4. 点击工具栏中的扩展图标打开 popup，或右键点击扩展图标打开侧边栏以自定义设置。
5. 玩得开心！

#### 网页应用

1. 访问 [托管网站](https://badwriter.ibuprofennist.com/).
2. 没别的了。玩得开心！
