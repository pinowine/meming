# Privacy Policy

This privacy policy applies to the BadWriter Chrome extension (hereinafter referred to as "the Extension").

### 1. Data Collection

The Extension highly respects user privacy. **The Extension does NOT collect, store, transmit, or share any personally identifiable information (PII) or browsing history.**

### 2. Data Usage

All core functionalities of the Extension (text fetching, text rewriting, and animation rendering) process data **strictly offline and locally in your browser**. The webpage text you select or hover over is only processed temporarily in your device's memory to perform the intended visual changes and is NEVER sent to any external servers or third parties.

### 3. Local Storage

The Extension utilizes the native Chrome Storage API (`chrome.storage.local`) solely for saving:

- Your custom text replacement rules.
- The active/inactive state of the Extension.
  This data resides exclusively on your personal device. You can clear or modify this data at any time via the Extension's interface or browser settings.

### 4. Third-Party Services

The Extension does not use any third-party analytics, tracking, or remote telemetry services.

### 5. Policy Updates

If there are any changes to our permissions or data handling practices in the future, we will update this privacy policy accordingly and detail the changes in the Chrome Web Store release notes.

### 6. Contact Us

If you have any questions, concerns, or requests regarding this Privacy Policy or our data handling practices, please contact the developer via the official GitHub repository issue tracker or your preferred contact email.
